
Goal: Getting an overview of the different dimensions of AI from a solution architect's point of view.
AI has several orthogonal dimensions:


1. Learning paradigm (how it learns?)
- Supervised
- Unsupervised
- Self-supervised
- Reinforcement Learning

2. Model architecture (what it is built with?)
- Linear models 
- Tree models
- Neural networks (CNN, RNN, transformer)

3. Application domain (where it is used?)
- Computer vision
- NLP
- Speech
- robotics
- time series

4. Capability (what it does?)
- predictive AI:
	- classification (output type)
		- binary (2 classes: spam / not spam)
		- multi-class (N classes, pick 1: cat / dog / bird)
		- multi-label (N classes, pick many: action + comedy + romance)
	- regression (output type)
		- single output (predict one number: house price)
		- multiple output (predict several numbers: lat + lng)
- generative AI (creates new content, text, images, videos, speech)
- decision making AI (agents, RL systems)
